# Hearthvale generated gap-fill art (original, committable)

Original procedural top-down tiles + simple UI shapes for the `sprout_topdown` visual mode, generated by `tools/art/generate_hearthvale_gap_assets.py`. NOT derived from Sprout Lands or any third-party media. These are the GENERATED fallback the registries resolve when no licensed asset is active, so a clean checkout (no Sprout pack) still reads as a coherent top-down farming world.
