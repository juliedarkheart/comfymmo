# New player onboarding

## The intended path

spawn at Hearthvale Landing (Rowan's farm) → read the welcome board → talk to
Farmer Rowan (she gives a **Land Token** and the first instructions) →
hand-gather branches/pebbles/fiber/clay → craft starter tools (K) → chop
trees / mine boulders / farm with tools → craft components at the workbench →
claim a neighborhood lot at a plot sign → build on it → invite friends (soon).

Rowan's dialogue is **staged on real progress**: no axe → she explains hand
crafting; tools but no land → she points at the plot signs; landholder → she
talks construction. The mailbox tasks remain the structured task list and now
pay material + stewardship-XP rewards. (A full quest engine stays deferred.)

## Starter soft-lock prevention (the guarantee)

A brand-new player with an EMPTY inventory can always reach full tooling:

1. Every raw material has a **hand-gatherable** source requiring no tool:
   fallen branches (wood), pebbles (stone), wild fiber, soft clay — validated
   to exist in the spawn registry for all four.
2. Every starter tool recipe is **level 1, no station, raw materials only**
   (e.g. worn axe = 2 wood + 2 stone) — validation rejects any starter recipe
   that requires a station, a level, a component, or another tool.
3. New profiles ALSO receive the full starter loadout (axe, pickaxe, hoe,
   watering can, hammer, shovel) on first boot/join — the crafting path is the
   safety net, the kit is the convenience.
4. Therefore no cycle ("wood needs axe needs wood") can ever strand a player:
   hand wood exists regardless of tools.

## Tool-gated jobs

Axe → chop trees · pickaxe → mine boulders · shovel → clay deposits + laying
paths/terrain · hoe → planting · watering can → watering · hammer → all other
building. Denials always name the tool and remind you K crafts it.

## What new players do NOT get

A free home. Houses are exterior shells you earn by gathering, crafting, and
claiming a lot — and interiors come later (docs/interiors_plan.md).
