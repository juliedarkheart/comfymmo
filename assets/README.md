# Assets

Hearthvale stores AI-assisted assets as reproducible bundles instead of loose
image dumps.

Key folders:

- `concepts/` for exploration and approved visual direction
- `sprites/` for character, prop, and building outputs
- `textures/` for terrain and material workflows
- `lora/` for future training datasets and checkpoints
- `workflows/comfyui/` for exported local generation graphs
- `imports/` for reviewed engine-facing files
- `published/` for current production-approved assets
- `metadata/templates/` for bundle manifest templates

Start with the rules in [docs/asset_pipeline.md](/D:/GitHub/comfymmo/docs/asset_pipeline.md).
